## 기능구현
### 기본 기능
- 로그인
- 회원가입
- (DB저장 여부 확인 후 존재하면 로그인 / 없으면 회원가입)
- 책 리스트 출력

### 구매 기능
- 책 리스트 출력
- 번호/이름 입력
- 구매대기 상태로 PURCHASE 테이블에 저장
- 내 구매내역에서 확인
- 구매확정 --> 구매완료 상태로 변경
- (책 수량(-), 포인트(-), 보유 책 리스트(+))
- 구매 리스트 출력
-
- 상태 업데이트(구매대기 / 구매완료)
- 

### 좋아요 기능 
- 책 번호/이름 입력
- LIKE_BOOK 테이블에 계정 아이디 + 책 아이디 저장
